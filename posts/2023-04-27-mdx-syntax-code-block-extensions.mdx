## Line Numbers

default config: `site.config.js` -> `config.markdown.lineNumbers`

**Input**

````mdx
```js lineNumbers={false}
// lineNumbers is disabled
const line2 = 'This is line 2'
const line3 = 'This is line 3'
```
````

````mdx
```js lineNumbers
// lineNumbers is enabled
const line2 = 'This is line 2'
const line3 = 'This is line 3'
```
````

**Output**

```js lineNumbers={false}
// lineNumbers is disabled
const line2 = 'This is line 2'
const line3 = 'This is line 3'
```

```js lineNumbers
// lineNumbers is enabled
const line2 = 'This is line 2'
const line3 = 'This is line 3'
```

## Line Highlighting in Code Blocks

**Input**

````mdx
```text lineNumbers highlights='1,3-4'
line 1
line 2
line 3
line 4
line 5
```
````

**Output**

```text lineNumbers highlights='1,3-4'
line 1
line 2
line 3
line 4
line 5
```

## Colored Diffs in Code Blocks

**Input**

````mdx raw
```js
const data = {
  message: 'Removed', // [!code --]
  message: 'Added', // [!code ++]
}
```
````

**Output**

```js
const data = {
  message: 'Removed', // [!code --]
  message: 'Added', // [!code ++]
}
```

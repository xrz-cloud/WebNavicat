## Line Highlighting in Code Blocks

**Input**

````mdx
```text highlights='1,3-4'
line 1
line 2
line 3
line 4
line 5
```
````

**Output**

```text highlights='1,3-4'
line 1
line 2
line 3
line 4
line 5
```

## Colored Diffs in Code Blocks

**Input**

````mdx raw
```js
const data = {
  message: 'Removed', // [!code --]
  message: 'Added', // [!code ++]
}
```
````

**Output**

```js
const data = {
  message: 'Removed', // [!code --]
  message: 'Added', // [!code ++]
}
```

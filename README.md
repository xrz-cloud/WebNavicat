# xiaojun1994.top

一个基于 Next.js 的开源博客

https://xiaojun1994.top/

## 特点

- 🎨 简约，丝滑，快速
- 🌚 支持黑暗模式，响应式设计适配移动端
- 🎮 支持 code 块的实时运行（原生 html 以及单个 react 组件）
- ⚡️ 由 Next.js + MDX + Tailwind CSS 强力驱动，方便二次开发，支持一键部署到 Vercel
- 🧩 集成不蒜子统计，预计阅读时长，文章标签分类
- 更多功能还在开发中...

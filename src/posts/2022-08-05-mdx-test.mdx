## HTML 实时预览

你可以实时编辑下方代码，右侧 demo 会同步更新

```html live
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Title</title>
    <style>
      body {
        margin: 0;
      }
      .circle {
        width: 100px;
        height: 100px;
        background: royalblue;
        border-radius: 50px;
      }
    </style>
  </head>
  <body>
    <div class="circle"></div>
  </body>
</html>
```

## React 组件实时预览

你可以实时编辑下方代码，右侧 demo 会同步更新 **（不支持多个组件）**

```jsx live
function Count() {
  const [count, setCount] = useState(0)

  return (
    <>
      <p>count: {count}</p>
      <button onClick={() => setCount(count + 1)}>+1</button>
    </>
  )
}
```
